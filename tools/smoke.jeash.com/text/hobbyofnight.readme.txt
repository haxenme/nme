The font "Hobby Of Night" is taken from:

	http://openfontlibrary.org/en/font/hobby-of-night

The .hash and .swf files are built using the hxswfml tool, see 
http://code.google.com/p/hxswfml/

To build the files:

	neko hxswfml.n ttf2hash hobbyofnight.ttf
	neko hxswfml.n ttf2swf hobbyofnight.ttf hobbyofnight.ttf.swf -class HobbyOfNight
